# Sistema de Gestión de Citas Médicas

## Descripción
Este es un proyecto académico desarrollado en **Java** utilizando el entorno **Apache NetBeans**. Consiste en una aplicación de escritorio para la administración eficiente de pacientes, médicos y citas, aplicando los principios de la **Programación Orientada a Objetos (POO)**.

## Características Técnicas️
* **Arquitectura:** Modelo basado en clases y objetos con separación de lógica.
* **POO:** Implementación de Herencia (`Persona` -> `Medico`), Polimorfismo e Interfaces (`Pagable`).
* **Gestión de Datos:** Uso de Colecciones (`ArrayList`) para manejo dinámico de registros.
* **Interfaz:** Diseño de GUI para interacción intuitiva.

## Diagrama de Clases (Estructura)
El sistema cuenta con una jerarquía robusta:
* **Clase Abstracta:** `Medico` (Define atributos base).
* **Interfaz:** `Pagable` (Calcula honorarios según especialidad).
* **Controlador:** `GestorCitas` (Maneja la lógica de negocio y sobrecarga de métodos).

## Tecnologías Utilizadas 
* Java (JDK 17 o superior)
* Apache NetBeans
* Git/GitHub
